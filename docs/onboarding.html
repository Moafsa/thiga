📚 Passos Detalhados

1️⃣
Criar Conta no Thiga
Configure sua conta no Thiga TMS e receba acesso administrativo.
O que você recebe:
✅ Tenant criado no sistema Thiga
✅ Plan associado (Trial, Pro ou Enterprise)
✅ Admin user com acesso total
✅ Dashboard e painéis de controle
✅ Link para integração WuzAPI
Informações importantes:
Dados de acesso:
  • 📧 Email: seu-email@empresa.com
  • 🔑 Senha: (temporária, será alterada no primeiro acesso)
  • 🏢 Empresa: Sua Transportadora
  • 🌐 URL: https://thiga.com.br/dashboard
✅ Status de sucesso: Você consegue acessar o dashboard e ver o menu principal.
Próximo passo:
→ Navegue para "Configurações" → "Integrações"
2️⃣
Configurar WuzAPI
Integre com WuzAPI para emitir CT-es e MDF-es (documentos fiscais eletrônicos).
O que é WuzAPI?
WuzAPI é o serviço especializado em emissão de documentos fiscais eletrônicos (CT-e e MDF-e). É como uma "fábrica de documentos" que se integra com a Sefaz (governo).
Passo a passo para criar conta WuzAPI:
  1. Acesse https://wuzapi.com.br
  2. Clique em "Criar Conta"
  3. Preencha os dados da sua empresa:
    • Razão Social exata
    • CNPJ (sem pontuação)
    • Email da empresa
    • Telefone para contato
  4. Aguarde aprovação (geralmente 1-2 horas)
  5. Receberá por email:
    • 🔑 API KEY (chave de acesso)
    • 🔐 API SECRET (código secreto)
Adicionar credenciais no Thiga:
⚠️ Importante: Mantenha a API KEY e API SECRET em segurança. Nunca compartilhe com terceiros.
✅ Status de sucesso: Ao clicar em "Testar conexão", você verá "Conectado com sucesso!"
3️⃣
Cadastrar Motoristas
Registre os motoristas que dirigirão os veículos da sua frota.
O que é um Motorista?
Um motorista é uma pessoa que dirigirá seus veículos e será responsável pelo transporte das mercadorias. Cada motorista pode dirigir múltiplos veículos.
Informações necessárias:
Onde cadastrar:
Menu → Gerenciamento → Motoristas → Novo Motorista
Exemplo prático:
Motorista 1: João Silva
• CPF: 123.456.789-00
• CNH: 9876543210 (válida até 2028)
• Email: joao.silva@empresa.com
• Telefone: (11) 98765-4321
• Status: ✅ Ativo
⚠️ Importante: Certifique-se de que a CNH é válida. CNH vencida causará rejeição de CT-es.
4️⃣
Cadastrar Veículos
Registre todos os veículos (caminhões, vans) que sua frota possui.
O que é um Veículo?
Um veículo é um caminhão, van ou qualquer meio de transporte que será utilizado para transportar mercadorias. Cada veículo será associado a um motorista.
Informações Essenciais:
Onde cadastrar:
Menu → Gerenciamento → Veículos → Novo Veículo
Por que essas informações?
Peso e dimensões: São usados para:
• Calcular frete
• Validar carga máxima
• Aparecer no CT-e (obrigatório para Sefaz)
• Ser usado em algoritmos de otimização de rotas
Exemplo:
Veículo: Volvo FH16
• Placa: PRF-1A23
• Peso Bruto: 25.000 kg
• Volume: 80 m³
• Motorista: João Silva ✅
5️⃣
Cadastrar Clientes
Registre empresas que enviam (remetentes) ou recebem (destinatários) mercadorias.
O que é um Cliente?
Um cliente é uma empresa ou pessoa que participa da operação. Pode ser:
🔸 Remetente: Quem envia a mercadoria
🔸 Destinatário: Quem recebe a mercadoria
🔸 Payer: Quem paga pelo frete
Informações necessárias:
Tipos de Cliente:
Pessoa Jurídica (PJ): Empresas
Pessoa Física (PF): Autônomos, proprietários

Categorias:
☑️ Fornecedor (envia mercadoria)
☑️ Cliente (recebe mercadoria)
☑️ Ambos
Exemplo de Remetente:
ABC Distribuição
• CNPJ: 12.345.678/0001-99
• Av. Paulista, 1000 - SP
• Email: vendas@abc.com
• Tipo: Fornecedor ✅
Menu → Gerenciamento → Clientes → Novo Cliente
6️⃣
Criar Rotas
Planeje o trajeto que o veículo fará, visitando múltiplas paradas.
O que é uma Rota?
Uma rota é um trajeto planejado que um veículo fará em um dia, visitando múltiplos clientes para coletar ou entregar mercadorias. Por exemplo: "Saída São Paulo 08h → Parada 1 em Guarulhos → Parada 2 em Campinas → Retorno 18h"
Componentes de uma Rota:
🚛 Veículo: Qual caminhão usará (ex: PRF-1A23)
👤 Motorista: Quem dirigirá (ex: João Silva)
📅 Data: Quando a rota será executada
📍 Paradas: Locais onde coleta/entrega (com ordem e horários)
Exemplo de Rota Completa:
ROTA-001-2024: São Paulo → Rio de Janeiro

🚛 Veículo: PRF-1A23 (Volvo FH16)
👤 Motorista: João Silva
📅 Data: 21/05/2024

Paradas:
📍 Parada 1 (08:00-09:00) - COLETA
    Empresa: ABC Distribuição
    Endereço: Av. Paulista, 1000 - São Paulo/SP
    Mercadoria: 330 camisetas - 165 kg

📍 Parada 2 (14:00-15:00) - ENTREGA
    Empresa: XYZ Varejo
    Endereço: Rua Central, 500 - Rio de Janeiro/RJ
    Mercadoria: 150 camisetas - 75 kg

📍 Parada 3 (16:30-17:30) - ENTREGA
    Empresa: Varejo Metropolitano
    Endereço: Rua das Flores, 200 - Duque de Caxias/RJ
    Mercadoria: 80 camisetas - 40 kg

📊 Resumo:
Total: 260 kg | Volume: 2,5 m³ | Distância: 450 km
Menu → Operações → Rotas → Nova Rota
⚠️ Importante: As coletas devem ser criadas ANTES da rota. As paradas serão adicionadas à rota ao criar a coleta.
7️⃣
Cadastrar Coletas
Registre cada carregamento de mercadoria a ser transportado.
O que é uma Coleta (Shipment)?
Uma coleta é um carregamento de mercadoria que sai de um remetente e vai para um destinatário. É o "pedido" ou "envio" que será transportado pelo veículo.
Informações da Coleta:
Informações de Mercadoria:
Para cada item da coleta, informar:
📦 Descrição: O que é (ex: Camisetas Azuis)
📊 Quantidade: Quantas unidades
⚖️ Peso: Peso em kg
📐 Volume: Volume em m³
⚠️ Natureza: Tipo (Carga Perigosa, Refrigerada, etc)
Exemplo Completo:
COLETA-2024-001

📦 Item 1: Camisetas Azuis
    Quantidade: 100 unidades
    Peso: 50 kg
    Volume: 0,5 m³

📦 Item 2: Camisetas Vermelhas
    Quantidade: 150 unidades
    Peso: 75 kg
    Volume: 0,75 m³

📦 Item 3: Camisetas Pretas
    Quantidade: 80 unidades
    Peso: 40 kg
    Volume: 0,4 m³

📊 TOTAIS:
Total Itens: 330 unidades
Total Peso: 165 kg
Total Volume: 1,65 m³
Menu → Operações → Coletas → Nova Coleta
8️⃣
Gerar CT-es
Sistema gera automaticamente documentos fiscais eletrônicos para cada coleta.
O que é um CT-e?
CT-e = Conhecimento de Transporte Eletrônico

É um documento fiscal obrigatório emitido pela Sefaz (governo) que comprova o transporte de uma mercadoria. É como uma "nota fiscal de transporte".
Como é Gerado Automaticamente?
Fluxo automático:

1️⃣ Coleta é criada e atribuída a uma rota
↓
2️⃣ Você atribui VEÍCULO + MOTORISTA à rota
↓
3️⃣ Sistema valida dados (CNH, RENAVAM, etc)
↓
4️⃣ Sistema gera CT-e automaticamente ✨
↓
5️⃣ CT-e é enviado para WuzAPI
↓
6️⃣ WuzAPI envia para Sefaz
↓
7️⃣ Sefaz valida e aprova 🟢
↓
✅ CT-e autorizado e pronto para uso!
Status do CT-e:
Status Significado O que fazer
🟡 PENDENTE Aguardando envio para Sefaz Sistema ainda está validando
🔵 VALIDANDO Validando dados Aguarde alguns segundos
🟣 PROCESSANDO Enviado à Sefaz, aguardando resposta Espere 5-10 minutos
🟢 AUTORIZADO ✅ Aprovado! Pronto para usar Pode transportar e imprimir
🔴 REJEITADO ❌ Sefaz rejeitou Verifique dados e corrija
⚪ CANCELADO Cancelado após aprovação Gere novo CT-e se necessário
Ver CT-es Gerados:
Menu → Fiscal → Documentos Fiscais

Lá você verá:
• Todos CT-es gerados
• Status de cada um
• Possibilidade de baixar PDF e XML
• Botão para cancelar (se necessário)
✅ Sucesso: Quando CT-e passar para status 🟢 AUTORIZADO, está pronto para transporte!
9️⃣
Gerar MDF-es
Consolida múltiplos CT-es em um único manifesto para a rota.
O que é um MDF-e?
MDF-e = Manifesto de Documento Fiscal Eletrônico

É um documento que consolida TODOS OS CT-es de uma rota. É como um "comprovante de que todos esses CT-es estão sendo transportados juntos no mesmo veículo".
Relacionamento CT-e vs MDF-e:
Uma Rota pode ter MÚLTIPLOS CT-es:

MDF-e ROTA-001 consolida:
├─ CT-e #001 (ABC → XYZ)
├─ CT-e #002 (ABC → Varejo)
└─ CT-e #003 (Distribuidor → Loja)

Resultado: 1 MDF-e que prova que todos 3 CT-es estão no mesmo caminhão na mesma rota
Quando é Gerado?
Automaticamente quando:
✅ Todos CT-es da rota estão 🟢 AUTORIZADO
✅ Rota tem VEÍCULO designado
✅ Rota tem MOTORISTA designado

Geração: Automática, geralmente alguns minutos após últim CT-e ser autorizado
Exemplo de MDF-e:
MDF-e ROTA-001-2024 - SÃO PAULO → RIO DE JANEIRO

🚛 Transporte:
  Veículo: PRF-1A23 (Volvo FH16)
  Motorista: João Silva
  Data: 21/05/2024

📄 CT-es Consolidados (3):
  ├─ CT-e #001 - ABC → XYZ (165 kg)
  ├─ CT-e #002 - ABC → Varejo Met. (180 kg)
  └─ CT-e #003 - Distribuidor → Loja (150 kg)

📊 Totais:
  Peso Total: 495 kg
  Volume Total: 5 m³
  Documentos: 3 CT-es
  Valor Total: R$ 99.750,00

🟢 Status: AUTORIZADO
Ver MDF-es: Menu → Fiscal → Documentos Fiscais → Filtro: MDF-e

🔧 Troubleshooting Rápido

❌ CT-e com status REJEITADO?

Causas comuns:

  • ❌ Motorista com CNH vencida
  • ❌ Veículo sem RENAVAM válido
  • ❌ Cliente com CNPJ inválido
  • ❌ Campos de endereço incompletos
  • ❌ Mercadoria sem descrição clara

Solução: Corrija os dados e tente novamente

⏳ CT-e PENDENTE por muito tempo?

O que fazer:

  1. Ir para Fiscal → Documentos Fiscais
  2. Clique no CT-e PENDENTE
  3. Botão "Reenviar"
  4. Aguarde 5-10 minutos
  5. Atualize a página (F5)

❓ MDF-e não foi gerado?

Verifique:

  • ✓ Todos CT-es estão 🟢 AUTORIZADO?
  • ✓ Rota tem VEÍCULO designado?
  • ✓ Rota tem MOTORISTA designado?
  • ✓ Coletas estão adicionadas à rota?

Nota: MDF-e é gerado automaticamente. Aguarde alguns minutos após todos CT-es serem autorizado.
