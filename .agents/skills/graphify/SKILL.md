# Graphify Knowledge & Execution Skill

This skill turns the TMS SaaS codebase and tenant database into a deterministic relational knowledge graph.

## Capabilities:
1. **System Node Inspection**: Trace relationships between Tenant -> User -> Driver -> Vehicle -> Route -> Shipment -> CteXml -> FiscalDocument -> Invoice.
2. **Predictive Analytics & Optimizations**: Evaluate routes, vehicle capacity, driver assignments, and CT-e utilization.
3. **Automated System Actions**: Execute CRUD operations (create routes, register drivers, vehicles, clients, delete entities) directly using Graphify AI Assistant (`GraphifyAiController`).
